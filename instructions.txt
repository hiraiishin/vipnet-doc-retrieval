Инструкция по запуску

Код проекта реализован в виде одного Jupyter Notebook и не требует сложной настройки окружения.

Для запуска необходимо:

Установить Python версии 3.10 или выше.

Установить Jupyter Notebook

Разместить PDF-документы в директории data/ (относительно корня проекта).

Все основные шаги обработки данных, построения индекса, выполнения retrieval и проведения экспериментов реализованы последовательно в ноутбуке. Код можно запустить целиком с помощью команды Run All.

Необходимые сторонние библиотеки (включая pdfplumber, sentence-transformers, faiss и другие) устанавливаются непосредственно в ноутбуке через соответствующие ячейки с pip install.
В случае отсутствия стандартных библиотек Python они также могут быть установлены аналогичным образом.

Дополнительных ручных действий для воспроизведения результатов не требуется.
