Инструкция по запуску проекта

Проект реализован в виде одного Jupyter Notebook и предназначен для демонстрации пайплайна поиска по технической документации в формате PDF.

Структура проекта

notebook.ipynb — основной ноутбук с кодом парсинга PDF, построения retrieval и экспериментами;

data/ — директория для исходных PDF-документов;

parsed/ — директория для результатов парсинга и разбиения документов в формате JSONL;

report.pdf — отчёт с описанием решения и результатов.

В папках data/ и parsed/ отсутствуют отдельные файлы README, так как их назначение описано в данном документе и в отчёте.

Запуск

Установить Python версии 3.10 или выше.

Установить Jupyter Notebook или Jupyter Lab.

Поместить PDF-документы в директорию data/.

Удалите заглушки в папках parsed и data

Открыть файл notebook.ipynb.

Выполнить все ячейки последовательно с помощью команды Run All.

Все основные этапы обработки данных, построения индексов, выполнения поиска и проведения экспериментов реализованы непосредственно в ноутбуке.

Зависимости

Список используемых библиотек приведён в файле requirements.txt.
Кроме того, в ноутбуке присутствуют ячейки для установки необходимых зависимостей с помощью pip, что позволяет запустить код даже в случае отсутствия части библиотек в окружении.

Примечания

Большие PDF-файлы и полные JSONL-выборки не включены в Git-репозиторий и должны быть предоставлены отдельно.

Код не требует дополнительной конфигурации и рассчитан на воспроизведение в стандартном локальном окружении.
